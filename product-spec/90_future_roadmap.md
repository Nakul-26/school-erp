# 90 - Future Roadmap

This document outlines upcoming system enhancements and modules designed to extend the ERP beyond core MVC pages.

---

## 📅 Phase 1: Interactive Timetable Builder UX
*   **Dynamic Drag-and-Drop Grid**: Weekly calendar UI enabling visual room/teacher slot allocations.
*   **Collision Detection**: Visual alert markers triggered when a teacher or physical classroom is assigned to overlapping periods.
*   **Substitution Module**: One-click allocation of substitute teachers based on period workload algorithms.

---

## 📬 Phase 2: Alert System & Messaging Engine
*   **Multi-Channel Notifications**: Real-time push updates, automated SMS messages, and emails triggered by fee reminders, marks releases, or attendance warnings.
*   **Trigger Builders**: Custom criteria triggers (e.g. "Send SMS warning if student has 3 consecutive absences").
