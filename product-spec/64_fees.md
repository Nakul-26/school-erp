# 64 - Fees Module Specification

This document details the configuration of fee structures, invoices, payment collection, discount policies, and ledgers.

---

## 🎯 1. Business Goals
*   Automated fee ledgers setup on admissions or promotions.
*   Waiver rules application.
*   Printable receipts generation.

---

## 📋 2. Key Interface Views
*   **Categories Setup**: Tuitions, Transports, Lab fees.
*   **Cashier Portal**: Quick Student Search ➔ invoice checkboxes ➔ Payment method selection ➔ payment transaction.
*   **Receipt Cards**: Instant PDF receipts print layouts.
